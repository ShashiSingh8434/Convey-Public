# Convey

A modern real-time communication platform built with Flutter, Firebase, Cloudinary, and WebRTC.

Convey combines messaging, media sharing, push notifications, presence tracking, and peer-to-peer audio/video calling into a single scalable communication ecosystem.

Designed with a feature-first architecture and powered by Riverpod, Firestore, Realtime Database, Cloudinary, Firebase Cloud Messaging, and WebRTC, Convey delivers a fast, responsive, and production-ready communication experience.

# Call Architecture

Convey uses WebRTC for peer-to-peer media streaming.

Workflow:

1. Caller creates call session
2. SDP Offer generated
3. Offer stored in RTDB signaling node
4. Receiver receives FCM incoming-call notification
5. Receiver accepts/rejects
6. SDP Answer generated
7. ICE candidates exchanged
8. Peer connection established
9. Audio/Video streams connected

Supported call states:

* Ringing
* Accepted
* Rejected
* Busy
* Missed
* Ended
* Completed
